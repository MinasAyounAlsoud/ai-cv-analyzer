# 🚀 AI CV Analyzer

A full-stack AI-powered CV analysis tool that evaluates resumes using OpenAI.

---

## 📸 Demo

![Demo](demo.gif)

---

## ✨ Features

- Upload PDF CV
- AI-powered skill extraction
- ATS Score calculation
- Education & Projects detection
- Modern React UI (TailwindCSS)

---

## 🛠️ Tech Stack

- React (Vite)
- TailwindCSS
- Flask (Python)
- OpenAI API
- MongoDB Atlas
